# **App Name**: Zenith Timer

## Core Features:

- Pomodoro Timer: Start, pause, and reset a Pomodoro timer with customizable session and break durations.
- Progress Animation: Display a circular progress animation that visually represents the time remaining in the current session or break.
- Customizable Durations: Allow users to set custom durations for work sessions and breaks.
- Audible Notifications: Provide desktop-style ticking and bell sound notifications at the end of sessions and breaks.
- Local Storage: Persist user settings and preferences in local storage.

## Style Guidelines:

- Primary color: Soft sky blue (#87CEEB) to evoke calmness and focus.
- Background color: Very light desaturated blue (#F0F8FF) to maintain a clean and unobtrusive look.
- Accent color: Muted lavender (#D8BFD8) for interactive elements, providing a gentle contrast.
- Body and headline font: 'Inter' for a modern, neutral feel suitable for both.
- Use rounded cards and soft shadows to create a minimalist and visually appealing layout.
- Implement subtle transitions and animations for a polished user experience.
- Employ a set of minimalistic icons, specifically those with rounded aesthetics.
